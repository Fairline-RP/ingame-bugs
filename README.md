# Fairline Bug Reports

Welcome to the public bug reporting repository for Fairline.

This repository is used to report ingame bugs, issues, and quality-of-life suggestions related to the Fairline server.

If something is broken, not working as expected, or causing errors ingame, please create an issue using one of the available templates.

---

## Report a Bug

Use **Bug Report** when something ingame is broken or not working correctly.

Good bug reports should include:

- What happened
- What should have happened
- Steps to reproduce the issue
- Screenshot or video if possible
- Any F8 or console errors
- Location, character name, vehicle, property, or job involved if relevant

The more details you provide, the easier it is for the team to reproduce and fix the issue.

---

## Suggest an Improvement

Use **QOL / Feature Request** if you have an idea that would improve the server experience.

This can include:

- Quality-of-life improvements
- UI improvements
- Gameplay improvements
- Small feature suggestions
- Changes that would make systems easier to use

---

## Do Not Post Exploits Publicly

Do not publicly post details about bugs that can be abused.

This includes:

- Money duplication
- Item duplication
- Vehicle duplication
- Permission bypasses
- Admin menu issues
- Security issues

Report those privately to Fairline staff.

---

## Not For Player Reports

This repository is only for bugs and suggestions.

Do not use this repo for:

- Player reports
- Staff reports
- Ban appeals
- General support questions
- Personal complaints

Use the correct Fairline Discord channels for those issues.

---

## Before Creating an Issue

Please check if the issue has already been reported.

If it already exists, you can add extra information, screenshots, or reproduction steps in the existing issue instead of creating a duplicate.

---

## Thank You

Every useful report helps make Fairline better, more stable, and more polished.

Thank you for taking the time to report issues properly.
